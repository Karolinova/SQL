# SQL solutions in PostgreSQL and MSSQL and their extensions

This repository contains SQL solutions implemented mainly in PostgreSQL and, in selected cases, in MS SQL Server.

It is a collection of queries and functions solving specific data-related problems, with a focus on:
- data manipulation (SELECT, UPDATE)
- conditional logic in SQL queries
- creating and using SQL functions

Each folder represents a separate problem and includes:
- a description of the problem
- a SQL-based solution

The repository demonstrates practical approaches to:
- solving practical SQL problems
- implementing conditional logic in SQL queries
- implementing conditional data updates

Technologies:
- PostgreSQL
- MS SQL Server
